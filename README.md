# Zombie Survivor

## Installation

Assurez-vous d'avoir Node.js installé.

Installez les dépendances :

```bash
npm i
```

Lancer le jeu :
```bash
npm start
```

## Comment jouer ?
### Déplacements

- Utilisez les flèches du clavier (haut, bas, gauche, droite).

### Combat
- Visez avec la souris.

- Tirez en cliquant avec le bouton gauche de la souris.

- Rechargez avec la touche R.
